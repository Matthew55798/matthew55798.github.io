# 政治

# 左与右

“左派 / 右派”最早来自 **18 世纪法国大革命时期的议会座位安排**：

- 支持**推翻旧制度、削弱王权、扩大平等**的人坐在议会左侧
- 支持**维护传统秩序、等级、宗教与财产权**的人坐在右侧

> 从一开始，它就不是“好坏之分”，而是改革 vs 稳定的分歧。左派试图**解构既有秩序**，右派试图**维护既有秩序**。
> 

左派更关注被现有制度压制的个体的集合关心打破制度从制度方面解决问题，可能会更关注经济问题，右派觉得制度是现实的演化，他们认为制度应该被维护，个人自由也应该被更多的维护，制度改变偏向集体会导致自由减少。
左右到分歧不是逻辑的产物，只是历史产物。
