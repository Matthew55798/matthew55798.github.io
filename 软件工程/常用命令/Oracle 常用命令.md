## 检查死锁

- **使用V$LOCK视图：** 通过查询V$LOCK视图来识别死锁情况。

```sql
SELECT * FROM V$LOCK WHERE BLOCK=1;
```

- **终止会话：** 根据分析结果，手动终止导致死锁的会话。

```sql
ALTER SYSTEM KILL SESSION 'sid,serial#';
```

通过这些步骤，可以有效地识别和处理Oracle数据库中的死锁问题。
